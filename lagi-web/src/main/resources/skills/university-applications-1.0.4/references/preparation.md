# 占卜准备指南

---

## 通用原则

### 任何解读之前
1. **明确目的**：有一个清晰的问题或关注领域
2. **保持开放**：不带刚性预期
3. **记录时间**：记下解读的确切日期/时间
4. **环境整洁**：选择安静、舒适的空间

### 心理准备
1. 静心：深呼吸几次
2. 专注：集中于你的问题
3. 诚心：以真诚的意图对待
4. 接受：对意外答案保持开放

---

## 各方法所需信息速查

| 方法 | 必需信息 | 可选信息 |
|------|---------|---------|
| 易经 / 六爻 | 明确问题 | 起卦时间 |
| 八字 | 出生年月日时 + 地点 | 性别 |
| 紫微斗数 | 出生年月日时 + 地点 | 性别 |
| 塔罗 | 一个清晰的问题 | 牌组偏好 |
| 西方星盘 | 出生日期 + 时间（精确到分钟）+ 地点 | — |
| 数字命理 | 出生日期 + 姓名 | — |
| 手相 | 双手清晰照片 | 年龄、性别 |
| 面相 | 正面清晰照片 | 年龄 |
| 风水 | 平面图/地址 + 朝向 | 居住者出生日期 |
| 奇门遁甲 | 排盘时间 + 问题 | 问事类型 |
| 梅花易数 | 问题 + 数字/时间/触发物 | — |

---

## 八字特别注意

| 项目 | 说明 |
|------|------|
| 出生时间 | 精确到时辰（2 小时），四柱命理以时辰划分 |
| 夏令时 | 要还原为标准时间 |
| 跨时区 | 注意出生地实际时区 |
| 子时 | 23:00-01:00，需确认早子时/晚子时 |

## 提问指南

### ❌ 错误方式
- 问模糊问题（"我会怎样？"）
- 一次问多个问题
- 期待绝对答案
- 忽略矛盾信息

### ✅ 正确方式
- 一次一个清晰问题
- "我应该……？" 或 "关于……我需要知道什么？"
- 接受指引而非预言
- 用解读获取洞察而非借口

---

## 各类问题示例

### 感情/关系
- "关于我目前的关系，我需要知道什么？"
- "我应该追求这段连接吗？"

### 事业/工作
- "什么职业路径适合我？"
- "什么时候适合求职？"

### 决策
- "在这个决定中我应该考虑什么？"
- "如果我选择 X，可能的结果是什么？"

### 通用指引
- "我现在需要知道什么？"
- "这个月/年我应该关注什么？"

---

## 重要提醒

1. **不是预言**：占卜显示趋势，不是固定未来
2. **可变**：能量会变化，结果也会
3. **配合**：解读与自由意志配合，而非对抗
4. **参考**：用指引做出明智选择

---

## 专业读者的红旗

| ✅ 好的迹象 | ❌ 危险信号 |
|------------|-----------|
| 清楚解释方法 | 保证特定结果 |
| 合理定价 | 推销昂贵套餐 |
| 愿意解释过程 | 拒绝解释方法 |
| 不做保证 | 声称超自然力量 |
| 有口碑/评价 | 索要过多个人信息 |
