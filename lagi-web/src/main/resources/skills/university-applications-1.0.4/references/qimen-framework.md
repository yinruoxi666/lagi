# 奇门遁甲解读框架

## 适用问题

奇门更适合看：
- 某件事现在适不适合推进
- 求职、面试、合作、沟通、谈判
- 见面、联系、复合窗口
- 近期阶段走势
- 该主动、试探，还是先守
- 方位吉凶

## 输入优先级

| 级别 | 条件 | 处理 |
|------|------|------|
| S 级 | 用户提供奇门盘截图或起局时间+问题背景 | 进入精读 |
| A 级 | 有准确问题+起念时间/起局时间 | 导向解读 |
| B/C 级 | 只有问题没有盘 | "奇门式阶段判断"，不假装具体门星神落宫准确 |

## 排盘计算

**严格按照 qimen-calculation-rules.md 中的公式和表格计算，绝不可凭印象估算。**

排盘七步流程：
1. 确定阴阳遁（冬至后阳遁，夏至后阴遁）
2. 定局数（查节气×旬首对照表）
3. 排地盘（局数对应天干为起点，阳顺阴逆）
4. 排天盘（值符星随旬首天干走）
5. 排人盘（值使门随时辰走）
6. 排神盘（值符所临宫位起，阳顺阴逆）
7. 标注中宫寄宫（阳遁寄坤二宫，阴遁寄艮八宫）

完整计算规则见：[qimen-calculation-rules.md](qimen-calculation-rules.md)

完整解读指南见：[qimen-interpretation-guide.md](qimen-interpretation-guide.md)

## 解读顺序

1. 先看整体局势：开、闭、阻、绕、藏、转
2. 再看用户位置：主动方、被动方、当前心态
3. 再看目标位置：事情、对方、机会、阻力
4. 再看动法：快进、慢推、绕行、暂避、借力
5. 最后看时间：近期窗口、暂缓期、观察期

## 写法要求

- 不要一上来堆"值符、值使、九星、八门、八神"
- 先翻成人话：这局到底是顺、堵、虚、乱、暗、转
- 真要提术语，也要顺手解释

## 常见结论模板

### 感情 / 复合
- "这局更像有念头，但不稳，联系可以试探，不适合逼结果。"
- "不是没有回应，而是对方心态飘，推进太猛反而关门。"

### 事业 / 合作
- "这件事能推进，但现在更适合铺垫，不适合当场拍板。"
- "机会不是没有，只是明面上看不到，要靠侧面切入。"

### 择时
- 用"适合联系、适合试探、适合收尾、适合观察"的语言
- 不写"某天不做就必凶"

## 禁止写法

- 没盘硬说盘面细节
- 把奇门当成绝对结果判决书
- 以奇门名义鼓励极端、违法、伤害性行为
- "此事必败""大凶不可动""会有血光之灾"
