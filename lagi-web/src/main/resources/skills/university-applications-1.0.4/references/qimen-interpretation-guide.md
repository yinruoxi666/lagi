## 四、常见奇门格局

### 吉格
- **青龙返首**：开门+天心星临乾六宫，谋事一帆风顺
- **飞鸟跌穴**：值符星与值使门同临一宫且为本宫，所求极易成功
- **三奇得使**：乙丙丁三奇同临某宫，文事武功均大利
- **天遁**：丙奇临生门或开门，阳气充盈如天助
- **地遁**：乙奇临开门或休门，柔中带吉
- **人遁**：丁奇临生门，人气旺盛利人际

### 凶格
- **白虎猖狂**：白虎神临开门或生门，凶神占据吉位
- **螣蛇夭矫**：螣蛇神临景门，文书不实需核实
- **大格（入墓格）**：天干落入所克宫位，事项受阻
- **返吟**：天盘与地盘完全相同，局势僵固
- **伏吟**：天盘各宫与本宫重叠，万事停滞
- **刑格**：天干地支相刑，被人制约或自我束缚

## 五、方位吉凶判断规则

**综合判断优先级**：
1. 八神（影响最大）：值符、六合、九天为吉；白虎、螣蛇、玄武为凶
2. 八门（次重要）：生门、开门、休门为大吉；死门、惊门、伤门为凶
3. 九星（辅助判断）：天心、天辅、天任、天冲为吉
4. 奇仪（基础加权）：乙丙丁三奇为吉；庚辛癸为凶

**方位综合评级**：
- 三层皆吉 → 大吉方位，可全力进取
- 两吉一凶 → 吉方，谨慎而行
- 一吉两凶 → 中平偏凶，不宜主动
- 三层皆凶 → 大凶方位，务必回避

## 六、表达风格

**开头范式**：
- "此盘{阳/阴}遁{X}局，值符{星}临{宫}，值使{门}临{宫}..."

**格局描述**：
- 吉格："{格局名}之象已现，{方位}方有贵气聚集，宜..."
- 凶格："{宫位}方有{格局名}之兆，此时此方向有一定阻力，建议..."

## 七、禁用表达

| 禁止使用 | 替换方案 |
|---------|---------|
| "此事必败/必成" | "此时格局对该事项{利/不利}，建议..." |
| "大凶，绝对不能动" | "此方位目前气场偏凶，宜暂缓" |
| "会有血光之灾" | "白虎临此宫，出行需注意安全" |
| "命中注定" | "当前时局显示..." |

绝对禁止：死亡预测、绝对化断言、恐吓性语言、推销改运服务。
