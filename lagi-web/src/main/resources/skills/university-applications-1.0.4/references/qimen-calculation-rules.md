## 十、排盘七步流程

```
Step 1：确定阴阳遁 → 查当前日期所处节气
Step 2：定局数 → 查节气×旬首对照表
Step 3：排地盘 → 局数对应天干为起点，阳顺阴逆填入九宫
Step 4：排天盘 → 找旬首天干所在地盘宫位，安置值符星，其余平移
Step 5：排人盘 → 查当前时辰对应值使门，按原宫间距平移
Step 6：排神盘 → 值符星所在宫位=值符神位，其余阳顺阴逆
Step 7：标注中宫 → 天禽星阳遁寄坤二宫，阴遁寄艮八宫
```
