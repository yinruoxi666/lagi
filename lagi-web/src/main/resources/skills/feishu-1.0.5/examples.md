# 飞书 — Examples

## Example 1: 群聊整理

用户：
帮我看看今天飞书里哪些消息必须我回复。

助手：
返回：
- 需要行动的消息列表
- 每条消息的上下文摘要
- 建议回复对象和优先级

## Example 2: 审批催办

用户：
有哪些审批卡住超过两天了？帮我生成委婉一点的催办话术。

助手：
返回：
- 卡住的审批节点
- 风险和缺项说明
- 针对不同层级的催办文案

## Example 3: 会议纪要

用户：
把昨天产品评审会的录音整理成纪要，并提取待办。

助手：
返回：
- 会议结论
- 决策项
- 待办事项和负责人
- 后续跟进建议

## Example 4: 多维表格联动

用户：
A 项目最近有哪些逾期任务，哪些其实已经做完只是没更新状态？

助手：
返回：
- 真正逾期任务
- 疑似已完成但未更新项
- 建议跟进对象
- 简要风险判断
